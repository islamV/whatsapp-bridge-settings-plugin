# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-05

### Added
- Initial release
- Multi-provider support: Bridge (self-hosted), Meta WhatsApp Cloud API, Twilio WhatsApp
- Filament 5 settings page with per-provider configuration tabs
- QR code pairing for the Bridge provider
- Connection status monitoring with polling
- Test messaging from the settings page
- Encrypted API tokens at rest
- OTP message templating with `{otp}` placeholder
- Bilingual support: English and Arabic translations
- Facade and Dependency Injection support
- Full test suite (58 tests)
